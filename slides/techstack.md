# Tech stack

The portfolio is built with a small, focused set of tools that keep the site fast, animated, and easy to maintain.

---

<!-- slide 2 -->

# Core app

- **Next.js** for the App Router, static deployment, and server components where needed
- **TypeScript** for predictable data flow and safer component props
- **Tailwind CSS** for layout, spacing, and visual consistency

---

<!-- slide 3 -->

# Motion and UI

- **Framer Motion** for scroll reveals, parallax, and section transitions
- **shadcn/ui** for accessible structural components
- **Magic UI** for decorative effects like particles, beams, and animated cards

---

<!-- slide 4 -->

# Data and deployment

- **GitHub API** for live profile and repository data
- **Seed data** for fallback content when APIs are unavailable
- **Static hosting** for simple deployment on Vercel or other hosts
