# User Feedback — Developer Portfolio

- **How collected:** Chat / 2-4 
- **When:** 30.6.26-9.7.26

## Raw feedback

1. " The intro could say more in fewer words."
2. "The GitHub section is useful because it feels live, but the layout gets a little busy."
3. "The tech stack and contact section are clear, but I had to scroll a lot before I found the part I wanted most. "

## Themes (what keeps coming up)

- The site feels strong visually, but the message could be more direct.
- Visitors want faster access to projects and proof of work.
- Some sections feel slightly dense when multiple animated elements appear at once.

## Top 3 things to fix

- Tighten the hero copy so the main value is clearer in the first screen.
- Reduce visual density in the busiest sections 
- Polish navigation.
